English-Avar Dictionary

This is an online English-Avar dictionary designed to help people learn and explore the Avar language (Maġarulał mацӀ) — a Northeast Caucasian language spoken primarily in Dagestan, Russia.

The dictionary uses a Latin-based transliteration system developed for avar.me, making it easier for international users to read and learn Avar words without needing to know the Cyrillic alphabet.

Source

This project is inspired by the lexicographic work compiled by Cyril Graham. The original materials and additional resources can be found at:

cg.avar.me

At the same time, the current website and the structured dictionary data are a product of LLM models Gemini 2.5 and Gemini 3.

Features

  • Search English words and find Avar translations
  • Instant search suggestions as you type
  • Both Latin and Cyrillic scripts displayed
  • Part of speech information for each word
